# Weekly investment committee

1. Read the week's daily reports and authoritative state.
2. Review market regime, earnings revisions, valuation dispersion, breadth, and leadership.
3. Re-rank all priority and candidate names using the Compounder Hunter scorecard.
4. Present the strongest bull and bear case for the top five.
5. Promote no more than three names to priority, each with valuation, expected return, invalidation, catalyst, and entry trigger.
6. Define next week's action map: buy-watch, wait, hold, reduce, invalidated, or no action.
7. Record any rating change in the append-only decision ledger.
8. Write `reports/weekly/YYYY-Www.md` using `reports/templates/weekly.md`.
