# Monthly decision audit

1. Select all decisions due for 30/90/180-day review.
2. Reconstruct the information available at the decision time. Do not import later facts into the original judgment.
3. Score thesis quality, evidence completeness, valuation, timing, sizing, invalidation discipline, and outcome.
4. Classify each decision:
   - right process / right outcome
   - right process / adverse outcome
   - wrong process / lucky outcome
   - wrong process / wrong outcome
5. Identify one or two recurring errors only when supported by multiple decisions.
6. Propose a strategy rule change with scope, evidence, expected benefit, and possible unintended consequence.
7. Add a new strategy version; never rewrite old versions.
8. Write `reports/monthly/YYYY-MM.md` using `reports/templates/monthly.md`.
