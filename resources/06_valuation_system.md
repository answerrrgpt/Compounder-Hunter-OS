# Valuation System

## Methods

- PE (contextual, not standalone)
- PEG ratio
- EV/EBITDA
- Free Cash Flow Yield
- DCF (scenario-based)
- Historical valuation percentile

---

## Key Principle
Valuation must be interpreted in context of growth and moat strength.

---

## Output Requirement

- Fair value range
- Upside scenario
- Downside scenario
- Current valuation status:
  - Undervalued / Fair / Overvalued / Bubble
